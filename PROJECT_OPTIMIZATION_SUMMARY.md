```

│   │   ├── css/
│   │   ├── js/

│       ├── base.html
│       ├── home.html

│       ├── user_management/
│       └── ...
├── readify/                    # Django应用目录
│   ├── books/                  # 图书管理应用
│   ├── ai_services/           # AI服务应用
│   ├── user_management/       # 用户管理应用
│   ├── tts_service/          # 语音合成服务
│   ├── translation_service/   # 翻译服务
│   └── settings.py           # 项目设置
├── media/                     # 媒体文件目录
├── staticfiles/              # 收集的静态文件
├── logs/                     # 日志文件
├── requirements.txt          # 项目依赖
└── manage.py                # Django管理脚本
```

## 📝 使用指南

```bash

# 启动开发服务器
python manage.py runserver 0.0.0.0:8000
```

- **主页**: http://localhost:8000
